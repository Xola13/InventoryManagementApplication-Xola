        Car Dealership Inventory Management Application

* My program consists of 4 Classes
- Car Extension class which allows you to add features to your cars
- Dealership class which contains all the methods that can be implemented on the application
- Car class this class contains and saves all the cars added to the application
- Program class this is where you are able to execute your application

* In this program application you are able to add a car with their required features
* Display all the available cars in the system 
* You can also Search a car by using their VIN.
